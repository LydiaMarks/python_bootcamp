Installation instructions for python_bootcamp:
    1. Forked original repo on Github: https://github.com/AllenInstitute/python_bootcamp/tree/main
    2. Cloned onto my local machine by doing: 
        $ (base) C:\Users\lydia.marks\code\Allen Python Foundations Course>git clone https://github.com/LydiaMarks/python_bootcamp.git
    3. copied environment.yml file from PythonMaterials and updated the name and prefix in the file to match python_bootcamp.
    4. created virtual environment using the environment.yml file
        $ conda env create -f environment.yml

to push changes from local to remote 
    1. if the main branch is ahead of my branch then go to github of my main forked repo and click sync fok. this will sync with the main fork.
    2. go to Git tab in VS code
    3. sync local branch with remote repo (circle made of arrows)
    4. inspect changes
    5. stage changes (press + )
    6. add a message in the box OR if you want multi-line messages, press commit and it will open up the file, where you can add a message at the top. if you do the file method, save the file and close it and then it will commit.
